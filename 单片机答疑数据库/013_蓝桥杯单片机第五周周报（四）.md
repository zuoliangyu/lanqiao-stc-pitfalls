### 数码管或按键无响应,卡死

* **问题描述**：
* **问题背景**：哔哩哔哩课程零基础部分，采用蓝桥杯单片机仿真
* **问题分析**：

  + 变量类型定义错误,使用unsigned char类型,范围为0-255
  + 变量范围错误,使用0-255范围,超过范围的变量值会被截断
* **解决方案**：

  + 使用unsigned int类型,范围为0-65535,替换掉错误的unsigned`char`类型

```

// 错误变量定义****************************************/

unsigned char Timer_1000ms; // 错误变量定义
unsigned char Seg_Slow_Down; // 错误变量定义
```

### （3）找不到hex文件问题

* **问题描述**：用户在STC-ISP软件中找不到hex文件进行烧录
* **问题背景**：哔哩哔哩课程零基础部分，初次使用单片机开发
* **问题分析**：

  + Windows系统默认隐藏文件扩展名，导致用户看不到.hex后缀
  + 用户生成的文件实际上是hex文件，但因为看不到后缀而识别错误
  + 文件显示设置问题，不是代码或编译问题
* **解决方案**：

  + 在Windows系统中设置显示文件名后缀
  + 操作步骤：打开文件资源管理器 → 查看选项卡 → 勾选”文件扩展名”
  + 设置后即可看到完整的文件名，包括.hex后缀

**操作步骤详解**：

1. 打开任意文件夹
2. 点击顶部”查看”菜单
3. 在”显示/隐藏”组中勾选”文件扩展名”
4. 重新查看项目文件夹，即可看到.hex文件

### （4）STC-ISP烧录软件使用问题

* **问题描述**：用户不知道如何正确使用STC-ISP软件进行程序烧录
* **问题背景**：哔哩哔哩课程零基础部分，首次接触单片机烧录
* **问题分析**：

  + 用户不熟悉STC-ISP软件的操作流程
  + 不清楚软件和硬件的操作顺序
  + 可能存在单片机型号选择或端口设置问题
* **解决方案**：

  + 掌握正确的烧录操作顺序
  + 先在软件中点击烧录按钮，再单击单片机上的下载按钮
  + 确保选择正确的单片机型号和COM端口

**正确操作流程**：

1. 打开STC-ISP软件
2. 选择正确的单片机型号
3. 选择COM端口（自动检测或手动选择）
4. 载入hex文件
5. **先点击软件中的”下载/编程”按钮**
6. **再单击开发板上的下载按钮**
7. 等待烧录完成提示

**注意事项**：

* 操作顺序很重要：先软件后硬件
* 确保USB连接稳定
* 如失败可重新插拔USB线缆

### （5）按键引脚定义不匹配问题

* **问题描述**：仿真环境下按键正常，但在实物开发板上按键无响应
* **问题背景**：哔哩哔哩课程零基础部分，从仿真过渡到实物开发
* **问题分析**：

  + 仿真软件和实际开发板的按键引脚定义不同
  + 仿真环境中使用P3.6、P3.7引脚
  + 实际开发板上按键连接到P4.0、P4.2引脚
  + 学生未及时更新引脚定义代码
* **解决方案**：

  + 修改代码中的按键引脚定义
  + 将仿真环境的P3.6、P3.7改为P4.0、P4.2
  + 在实物开发时注意检查原理图

**学习要点**：

* 仿真和实物开发存在硬件差异
* 需要根据实际硬件原理图修改引脚定义
* 养成查看硬件手册和原理图的习惯

### （6）未编译导致hex文件缺失问题

* **问题描述**：用户在项目文件夹中找不到hex文件进行烧录
* **问题背景**：哔哩哔哩课程零基础部分，Keil开发环境使用
* **问题分析**：

  + 用户编写代码后没有进行编译操作
  + hex文件是编译过程的产物，不编译不会生成
  + 用户可能保存了代码文件但跳过了编译步骤
* **解决方案**：

  + 在Keil软件中进行编译操作
  + 确保编译成功无错误
  + 在项目文件夹中查找生成的hex文件

**Keil编译步骤**：

1. 打开Keil工程文件
2. 检查代码是否有语法错误
3. 点击”编译”按钮（F7键）
4. 观察编译输出窗口，确保”0 Error(s), 0 Warning(s)”
5. 编译成功后，hex文件会自动生成在项目文件夹中

**常见编译问题及解决**：

* **语法错误**：根据错误提示修改代码
* **路径问题**：检查项目路径是否包含中文
* **配置错误**：检查Output选项卡中是否勾选”Create HEX File”

**hex文件位置**：

* 默认路径：项目文件夹\Objects\或项目根目录
* 文件名格式：项目名.hex
* 可在Keil的Project Options → Output中设置输出路径
