### （2）ucled赋值非0可以点亮（at89c52）

* **问题描述**：给ucled赋值一个任何的非零数都可以点亮
* **问题背景**：b站过渡视频
* **问题分析**：  
  查看底层逻辑
* **解决方案**：  
  因为底层只通过if判断真假进行点亮，所以非零的情况都一样。  
  // 关键代码示例

```
if(enable)
{
		temp|=0x01 << addr;
	else 
		temp&=~(0x01 << addr);--- 这里只判断enable，所以非零就可以点亮。
}
```
