### （2）时间判断逻辑出现问题

* **问题描述**：LED模式无法切换到模式2。
* **问题背景**：决赛试题-Led彩灯智能控制系统
* **问题分析**：

  + 直接定位到Led模式切换的逻辑找问题
* **解决方案**：

  + 这里使用了==比较，但System_Time是不断递增的，很难匹配到特定值。应该使用&gt;=比较。

### （3）在仿真中使用pdata出现错误

* **问题背景**：第二届创意智造·奠基未来 单片机设计与开发大赛 决赛

  + 在proteus中，默认pdata必须加上外扩ram才可运行

### （4）Led没有赋初始值出现了逻辑错误

* **问题描述**：大模板的LED模块，无法在仿真中运行
* **问题分析**：
* - 先判断LED能否独立点亮

  + 再到LED的逻辑中去找问题
  + 从模式0的前提条件是ucled==0x7e，发现ucled没有赋值
* **解决方案**：给予ucled初始值，程序正常运行

### （5)按键函数调用错误

* **问题描述**：按键无法切换数码管显示模式
* **问题分析**：

  + 将问题分解成按键和数码管切换逻辑
  + 先看按键是否有用，发现在Key_Val =Key_Read;中错误使用使用Key_Read函数
* **解决方案**：将Key_Val=Key_Read改为Key_Val=Key_Read()，后程序正常运行

```
  void Key_Proc()
  {
          if(Key_Slow_Down)return;
                  Key_Slow_Down =1;//按键减速程序
                  Key_Val =Key_Read;//读取按下的键码值
                  Key_Down =Key_Val &(Key_Val^Key_Old);//捕捉下降沿 按一下用这个
                  Key_Old =Key_Val;//辅助扫描 长按用这个
 
          switch(Key_Down)
  		{
                  case 1://跳到时钟设置界面，设置S1按键前要把时钟显示界面的值赋给设置界面才能停住
                  Clock_Set[0] =Clock_Disp[0];
                  Clock_Set[1] =Clock_Disp[1];
                  Clock_Set[2] =Clock_Disp[2];
            	  Seg_Disp_Mode = 1;
                  break;
  		}
  }
```
