### （1）按键无法控制流水灯的停止

* **问题描述**：按键可以控制流水灯的启动但无法控制流水灯的停止
* **问题背景**：[b站] [零基础入门二] [LED灯]，采用proteus仿真图
* **问题分析**：

  + 可能是因为用阻塞性延时导致按键无法实时检测
  + 这个程序除了delay都进行的太快了，点按键的那一步是在delay这个阶段时间内点的，无法奏效
* **解决方案**：

  + 用定时器实现非阻塞流水，主循环负责扫描按键控制启停，不用delay：

```
void key_proc()
{
    if(key_slow<=10) return;
    key_slow=0;
	Key_Val = Key_Read();
    Key_Down = Key_Val & (Key_Val ^ Key_Old);
   	Key_Up = ~Key_Val & (Key_Val ^ Key_Old);
    Key_Old = Key_Val;
	switch(Key_Down)
    {
         //流水灯启动
		case 1:System_Flag=1;break;
         //流水灯暂停
		case 2:System_Flag=0;break;
	}
}

void Timer0Server() interrupt 1
{
	TL0 = 0x18;
	TH0 = 0xFC;
	key_slow++;
    if(System_Flag==0) time_200ms=0;
    if(System_Flag==0) return;
    if(++time_200ms>=200)
    {
		time_200ms=0;
		usLed=_crol_(usLed,1);
		P1=usLed;
	}
}
```

### （3）数码管不闪烁

* **问题描述**：数码管无法正常闪烁
* **问题背景**：[b站] 电子钟] [数码管]，采用proteus仿真图
* **问题分析**：

  + 学院按照覆盖熄灭的写法，无法正常使选中的数码管闪烁
* **解决方案**：

  + 提供了三目运算符写法，使被选中的两位数码管进行闪烁，相关代码如下

```
// index为1 代表选择 时
// index为2 代表选择 分
// index为3 代表选择 秒

Seg_Buf[index*2]=(shanshuo)?(Time_Set_Val[i]/10 % 10):10;
Seg_Buf[1+index*2]=(shanshuo)?(Time_Set_Val[i] % 10):10;

//定时器
void Timer0Server() interrupt 1
{
	TL0 = 0x18;
	TH0 = 0xFC;
    if(++time_500ms>=500)
    {
		time_500ms=0;
		shanshuo^-1;
	}
}
```

* **问题背景**：[b站] [电子钟] [数码管]，采用proteus仿真图
* **问题分析**：

  + 学员用了三目却无法闪烁，先让学院将闪烁相关的直接扔while里，别的注释，进行排查

* **解决方案**：
