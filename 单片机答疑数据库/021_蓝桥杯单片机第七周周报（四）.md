### 数码管或按键无响应,卡死

* **问题描述**：
* **问题背景**：哔哩哔哩课程蓝桥杯模块部分，采用蓝桥杯单片机实物板
* **问题分析**：delay12ms使用的是stcisp生成的stc89的

```
void Delay12us(void) //@12.000MHz
{
	unsigned char data i;
	nop();
	i = 3;
	while (--i);
}
```

* **解决方案**：

```
void Delay12us(void) //@12.000MHz
{
	unsigned char data i;
	nop();
	nop();
	i = 38;//原本是33`，改为38，即延时12us
	while (–i);
}
```

### （2）按键循环检测

* **问题描述**：长按数据一直跳,短按正常。
* **问题背景**：哔哩哔哩零基础课程，按键长短按部分。
* **问题分析**：没有听西风的的长按检测模板,  
  乱用key_old导致,事件以很快的速度循环触发导致异常

* **解决方案**：
* 1,增加时间变量,和标志位控制按键以固定的时间和次数发生
* 2,保证每次触发只会发生一次
