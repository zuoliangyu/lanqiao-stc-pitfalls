### （1）程序卡死, 按键等多个功能无法使用

* **问题描述**：同时在 while(1)中使用 delay 和定时器导致程序卡死
* **问题背景**：哔哩哔哩课程零基础部分，采用蓝桥杯单片机仿真
* **问题分析**：

  + 避免 delay 的使用
  + 统一配置定时器进行不同时间段的配置
* **解决方案**：
* 只使用定时器函数,而避免delay的使用

```
// 关键错误代码示例
switch(mode)
{
    case 0:	
        P2=ucled1;
        Delay(500);
        ucled1=_crol_(ucled1,1);
        break;
    case 1:
        P2=ucled1;
        Delay(500);
        ucled1=_cror_(ucled1,1);
        break;
    case 2:
        P2=Led1[i];
        Delay(500);
        i++;
        if(i==4)i=0;
        break;
    case 3:
        P2=Led1[i];
        Delay(500);
        i--;
        if(i==255)i=3;
        break;
}
```

### （2）部分数码管闪烁写法错误, 从而导致现象错误

* **问题描述**：数码管闪烁使用了老的覆盖 Seg_Buf 的写法, 导致仿真和实物与本该有的现象不符
* **问题背景**：哔哩哔哩零基础课程，按键长短按部分。
* **问题分析**：

  + 数码管闪烁使用了老的覆盖 Seg_Buf 的写法
* **解决方案**：

避免数码管显示字符的覆盖, 保证一个状态下, Seg_Buf [i] 只会进行一次赋值, 而不是通过多次赋值来实现闪烁
