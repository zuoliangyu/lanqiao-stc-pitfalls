## 三、本周重点问题详解（可用图片）

---

### （1）变量类型使用错误(2)

* **问题描述**：输出和电压值不相等
* **问题分析：**

  + 查看Seg_Proc()发现强制转换的类型有错误
* **解决方案**：把unsigned char改成unsigned int类型

---

### （2）for循环无效

* **问题描述**：用for循环进行0-9的赋值

* **问题分析**：

  + 发现是for循环后面加了一个；使得{}内的内容没有被循环执行
* **解决方案**：

  + 把延时改成定时器去延时，而不是delay去堵塞程序

### （3）按键有问题

* **问题描述：S14按键长按累加变慢，S15按键无效
* **问题分析**：

  + 直接查看S14按键逻辑
  + 发现S14的up逻辑忘记添加{},导致
* **解决方案**：在s14的up添加{}

```
  //修改前
  if(Key_Up==14)
{
  	Timer_Flag=0;
  	Count_580ms=0;

}
  //修改后
  if(Key_Up==14)
  {
  	Timer_Flag=0;
 	 Count_580ms=0;
  }
```
