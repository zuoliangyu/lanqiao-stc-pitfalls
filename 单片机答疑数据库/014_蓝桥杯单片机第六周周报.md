### （1）按键按下无反应

* **问题描述**：按键按下后程序无任何响应，按键读取函数返回值异常。
* **问题背景**：B站第零讲第一题倒计时项目，使用51单片机。
* **问题分析**：

  + 按键读取函数中局部变量未初始化
  + 变量temp可能包含随机值
  + 导致按键判断逻辑错误
* **解决方案**：

  + 在按键读取函数中为局部变量temp赋初值0，修改代码如下：

```
unsigned char key_read()
{
    unsigned char temp = 0; // 局部变量赋初值0
    if(P3_4 == 0) temp = 1;
    if(P3_5 == 0) temp = 2;
    if(P3_6 == 0) temp = 3;
    if(P3_7 == 0) temp = 4;
    return temp;
}
```

### （2）闹钟响了之后LED灯常亮

* **问题描述**：电子钟项目中，闹钟到达设置时间后，LED灯一直保持常亮状态，无法正常控制。
* **问题背景**：电子钟
* **问题分析**：

  + 到达闹钟设置时间后，seg_mode变量没有被正确更改
  + LED控制逻辑依赖于seg_mode的状态
  + 状态机转换不完整，导致LED状态异常
* **解决方案**：

  + 在闹钟触发时，需要将seg_mode更改为合适的值，确保后续LED控制逻辑能够正常执行。

### （3）数码管显示滑动变阻器分压问题

* **问题描述**：在数码管只显示滑动变阻器分压的情况下，改变光照强度时数码管数值不发生变化。
* **问题背景**：无具体背景
* **问题分析**：

  + 光敏电阻和滑动变阻器是两个独立的硬件模块
  + 它们只是共用同一个单片机芯片
  + 各自的ADC通道和处理逻辑相互独立
* **解决方案**：

  + 两个硬件模块互不影响，这是正常现象。如果需要显示光敏电阻的数据，需要切换到对应的ADC通道和显示模式。

### （4）DA转换测电压引脚问题

* **问题描述**：进行DA转换时，不知道应该从开发板的哪个引脚测量输出电压。
* **问题背景**：无具体背景
* **问题分析**：

  + 开发板上DA转换器的输出引脚位置不明确
  + 缺乏硬件原理图或引脚说明的参考
  + 需要找到正确的测量点
* **解决方案**：

  + 在开发板上找到标有”D/A”标识的引脚或接口，这就是DA转换器的输出引脚，可以从该处测量电压。

### （5）蜂鸣器和继电器不能同时使能

* **问题描述**：蜂鸣器和继电器无法同时工作，只能单独控制其中一个。
* **问题背景**：无具体背景
* **问题分析**：

  + 两个外设控制函数各自使用独立的临时变量
  + 缺乏全局状态管理
  + 后执行的函数会覆盖前一个的设置
* **解决方案**：

  + 使用全局变量保存外设状态，实现状态累加控制：

```
unsigned char temp;
void Beep(unsigned char flag) // 蜂鸣器  
{  
    if(flag)  
        temp |= 0x40;  
    else  
        temp &= ~0x40;  
    if(temp != temp_old)  
    {  
        P0 = ~temp;  
        P2 = P2 & 0x1f | 0xa0;  
        P2 &= 0x1f;  
        temp_old = temp;          
    }  
}  

void Relay(unsigned char flag) // 继电器  
{  
    if(flag)  
        temp |= 0x10;  
    else  
        temp &= ~0x10;  
    if(temp != temp_old)  
    {  
        P0 = ~temp;  
        P2 = P2 & 0x1f | 0xa0;  
        P2 &= 0x1f;  
        temp_old = temp;          
    }  
}
```

### （6）按下按键不进行倒计时

* **问题描述**：按键按下后倒计时功能不执行，程序逻辑判断异常。
* **问题背景**：无具体背景
* **问题分析**：

  + 按键判断条件中使用了赋值运算符（=）
  + 应该使用比较运算符（==）
  + 导致条件判断逻辑错误
* **解决方案**：

  + 将按键判断条件中的单个等号改为双等号，确保正确的比较操作。

---

### （7）ADC转换时光敏电阻显示异常

* **问题描述**：使用ADC转换时，单独的光敏电阻可以正常显示数据，但与滑动变阻器一起显示时，滑动变阻器正常而光敏电阻无法显示。
* **问题背景**：无具体背景
* **问题分析**：

  + 多通道ADC切换时序问题
  + 可能存在通道切换延时不足
  + ADC转换完成标志处理不当
  + 数据读取函数调用顺序或频率问题
* **解决方案**：

  + 为光敏电阻添加独立的ADC读取函数，确保每个通道都有足够的转换时间和正确的数据读取流程。
